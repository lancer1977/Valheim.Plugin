# Valheim.Plugin Archive

`Valheim.Plugin` is archived and reference-only.

Active Valheim work has moved to `Valheim.RestServer`.

## What remains here
- legacy RPC and HTTP command experiments
- Harmony/Jotunn implementation references
- historical documentation for old plugin behavior

## What does not remain here
- no new public REST contract work
- no new stream-facing behavior
- no new feature development unless this repo is explicitly unarchived

## Canonical successor
- `~/code/Valheim.RestServer`

If you are looking for the current Valheim REST contract, use `Valheim.RestServer` instead of this repo.
