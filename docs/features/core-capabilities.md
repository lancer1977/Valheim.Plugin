# Valheim.Plugin Core Capabilities

The `Valheim.Plugin` foundation empowers developers with a standardized approach to building robust Valheim mods.

## Beyond the App

The value of `Valheim.Plugin` extends beyond simple gameplay modifications. It establishes a framework for professional-grade modding:

- **Remote Administration**: The modular architecture and RPC command system allow for the development of remote management tools that can interact with the game server from external platforms.
- **Architectural Reference**: It serves as a pedagogical resource for how to apply enterprise software patterns (like SOLID principles and unit testing) to the unique environment of Unity-based game modding.
- **Interoperability Framework**: By providing standardized hooks and a centralized command map, it facilitates the creation of ecosystems where different mods can communicate and work together harmoniously.
